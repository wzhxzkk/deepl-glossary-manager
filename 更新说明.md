### 3. ✅ scripts 文件夹说明

**用途**:
```
scripts/
├── README.md          # 说明文档
└── export_terms.py   # 导出术语表工具（示例）
```

**scripts vs 根目录脚本**:

| 位置 | 类型 | 说明 |
|------|------|------|
| 根目录 | 核心功能 | `glossary_manager.py`, `test_glossary.py` |
| scripts/ | 辅助工具 | 可选的便利脚本 |

**示例: export_terms.py**
- 功能: 导出 TERMS 字典为 JSON/TSV/Markdown
- 用途: 备份、分享、版本控制
- 使用: `python scripts/export_terms.py --format json`

## 🎯 为什么这样设计？

### 1. 中文优先
- 目标用户：中文学术研究人员
- 使用场景：阅读中文论文、管理 Zotero 文献
- 效果：打开 GitHub 直接看懂，无需切换语言

### 2. 保留英文支持
- 国际用户也可能使用
- 显眼的切换按钮（绿色徽章）
- 完整的英文文档

### 3. scripts 文件夹
- 核心功能不变（glossary_manager.py, test_glossary.py）
- 额外的便利工具放 scripts/
- 用户可选择性使用，保持主目录简洁

### 4. 个人笔记排除
- RELEASE_GUIDE.md → 只给你看的发布步骤
- PROJECT_SUMMARY.md → 项目总结，内部文档
- 项目完成报告.md → 给你的交付报告
- 这些不需要出现在公开仓库
